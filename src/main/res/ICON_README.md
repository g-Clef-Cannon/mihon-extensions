# Icon Placeholder

You need to add icon files for the extension. Place PNG files named `ic_launcher.png` in each of these directories:

- src/main/res/mipmap-mdpi/ (48x48)
- src/main/res/mipmap-hdpi/ (72x72)
- src/main/res/mipmap-xhdpi/ (96x96)
- src/main/res/mipmap-xxhdpi/ (144x144)
- src/main/res/mipmap-xxxhdpi/ (192x192)

For now, you can use the comic's avatar from: https://img.comicfury.com/avatars/43854.png

Resize it to the appropriate dimensions for each mipmap folder.
